# AI 에이전트 모드 — 사용자 시나리오 체크리스트

현재 구현(질문 모드 + 실행 모드 MVP/Phase 2)으로 가능한 사용자 시나리오를 자연스러운 흐름으로 정리.
2026-06-23 full-stack(자동임포트 복구·실데이터) 브라우저로 전 항목 검증. ✅ = 검증 완료.

## 질문 모드 (`/api/ai/chat`)
- [x] 평균 거래가 등 데이터 질의 — 예: "강남구 2024년 5월 평균 거래가 알려줘" → "23억 9,139만원" ✅

## 실행 모드 (`/api/ai/agent`) — 검색·필터
- [x] 지역+월 검색 — "강남구 2024년 5월 검색해줘" → 317건 + 지도 마커 ✅
- [x] 기간 범위 — "마포구 2024년 3월부터 6월까지 보여줘" → startDealMonth 2024-03 / endDealMonth 2024-06 ✅
- [x] 아파트명 — "강남구에서 래미안 검색해줘" → aptName=래미안 ✅
- [x] 가격대 — "5억에서 9억 사이만 보여줘" → minPrice 50000 / maxPrice 90000(만원) ✅
- [x] 정렬 — "비싼 순으로 정렬해줘" → sort=priceDesc ✅
- [x] 읍면동 — "강남구 역삼동 검색해줘" → umdNm=역삼동 ✅

## 실행 모드 — 결과 탐색
- [x] 항목 선택(상세) — "첫 번째 매물 선택해줘" → 상세 카드 + 지도 마커 강조/줌 ✅
- [x] 지도 포커스 — "세 번째 매물 지도에서 보여줘" → mapFocus(itemIndex=3) ✅
- [x] 페이지 이동 — "다음 페이지 보여줘" / "3페이지로" → paginate(direction/page) ✅
- [x] 경계 안내 — 마지막 페이지에서 "다음 페이지" / 범위 밖 항목 → 친화 안내(무크래시) ✅

## 실행 모드 — 제어·안전
- [x] 초기화 — "검색 조건 초기화해줘" → reset ✅
- [x] 서울 외 지역 → 되묻기 — "부산 해운대 아파트 검색해줘" → clarify "부산 지역은 지원하지 않습니다..." ✅
- [x] 모호한 요청 → 되묻기 — 조건 없는 "검색해줘" → clarify ✅

## 알려진 메모 (자연스러움 / 한계)
- "검색해줘/보여줘"가 가끔 `search` 대신 `setFilters`로 매핑됨(필터는 정확히 채워지나 검색 미실행).
  기능 결함은 아니며, 프롬프트 튜닝(검색·보여줘 동사를 search로 유도) 후보.
- 정렬은 지역(자치구)이 설정됐을 때만 적용(미설정 시 최신순 복원, 요약도 정렬 미주장).
- 가격 슬라이더/읍면동 목록은 지역·범위 로드 후 표시(검색 요청 값은 즉시 전송되어 결과는 정확).
- 항목/페이지 조작은 "현재 검색 결과" 기준(먼저 검색 필요).

## 지원 범위
- 지역: 서울특별시 25개 자치구(비서울 → clarify) · 기간: 2006년~현재(`YYYY-MM`) ·
  가격: 만원 단위 정수(min ≤ max) · 정렬: latest/oldest/priceDesc/priceAsc.

## 아직 불가 (백로그 P2)
- 다중 자치구 비교 · 추세/통계·면적·연식 질의 · 멀티턴 대화 기억(Phase 5) · 비서울 지역.
